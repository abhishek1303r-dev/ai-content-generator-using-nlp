# ai-content-generator-using-nlp
AI Content Generator Using NLP and SVM

1. Overview

AI Content Generator is a Machine Learning and NLP project that generates topic-based content in different tones. The application uses TF-IDF Vectorization and a Support Vector Machine (SVM) model to predict and generate relevant content based on user input.

This project demonstrates the complete NLP workflow, including text preprocessing, feature extraction, model training, and content prediction.

2. Features

Generate content based on topic and tone
NLP-based text processing using TF-IDF
Machine Learning model using SVM
User-friendly input system
Easy to expand with larger datasets
Beginner-friendly project structure

The dataset contains multiple topics and tones that help the model learn content patterns.

5. How It Works
Load the dataset from CSV.
Combine Topic and Tone into a single input feature.
Convert text into numerical features using TF-IDF.
Train an SVM model on the processed data.
Accept user input.
Generate content predictions based on the learned patterns.
